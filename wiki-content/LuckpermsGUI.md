# LuckpermsGUI Portal Hub

`LuckpermsGUI` provides server owners and staff with a full chest-GUI administrative front end for
LuckPerms — permission nodes, groups, promotion tracks, contexts, and the audit log, all managed
through in-game inventory menus instead of raw `/lp` command syntax.

## Resource Directory

* 📘 **[Technical Wiki & Configuration Reference](/LuckpermsGUI/wiki)**  
  Complete command registry, the `luckpermsgui.admin` permission node, the module-by-module GUI
  reference, and the chat-input syntax used for adding permissions, groups, and meta.

* 🎓 **[Step-by-Step Guides & Examples](/LuckpermsGUI/learn)**  
  Installation walkthrough, building a promotion ladder from scratch, and the operational gotchas
  worth knowing before you rely on this in production.

* 📥 **[Production Binary Downloads](/LuckpermsGUI/download)**  
  Grab the latest compiled stable release of `LuckpermsGUI.jar` directly from the host storage
  block.
